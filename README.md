# Discord PengUinBot (음악 봇)
<br/><br/>
## 명령어 목록
#### -join
> 봇을 통화방에 입장시킵니다.

<br/>

#### -leave
> 봇을 통화방에서 퇴장시킵니다.

<br/>

#### -clear
> 서버의 재생 목록을 초기화합니다.

<br/>

#### -play ``[노래 제목]``
> ``[노래 제목]`` 에 해당되는 노래를 재생 목록에 추가하거나 노래를 재생시킵니다.

<br/>

#### -queue
> 서버의 재생 목록을 확인합니다.

<br/>

#### -shuffle
> 서버의 재생 목록을 섞습니다.

<br/>

#### -skip
> 현재 재생 중인 노래를 스킵하고 다음 노래를 재생시킵니다.

<br/>

#### -nowplay ``[번호]``
> 재생 목록에서 ``[번호]`` 에 해당하는 노래를 재생시킵니다. (번호가 공백일 시 현재 재생 중인 노래를 보여줍니다.)

<br/>

#### -remove ``[번호]``
> 재생 목록에서 ``[번호]`` 에 해당하는 노래를 삭제합니다.

<br/>

#### -loop
> 재생 목록을 반복시킵니다.

<br/>

#### -playlist ``[저장/불러오기/확인]``
> 개인 재생목록을 저장하거나 불러옵니다.
